# Code Navigation Skill

Use MCP tools from codegraph mcp if possible

## Mandatory

When working with ANY code:

1. find_symbol
2. find_usages
3. analyze call graph
4. identify dependencies

## Workflow

1. Locate entry point
2. Trace call chain
3. Identify affected components
4. Modify only after understanding

## Anti-pattern

- guessing architecture
- editing without context
